# PlayBox - Application de Jeux de Société

Application mobile de jeux de société jouables sur un seul téléphone partagé entre joueurs.

## 🎮 Jeux disponibles

L'application propose une collection de jeux parfaits pour vos soirées :
- **Bombe** : Ne laissez pas la bombe exploser entre vos mains !
- **Action ou Vérité** : Le classique indémodable avec différents niveaux.
- **Tu préfères** : Des choix impossibles pour des débats enflammés.
- **Undercover** : Trouvez l'imposteur caché parmi vous.
- **Time's Up** : Faites deviner des mots en 3 manches.
- **Taboo** : Faites deviner des mots sans prononcer les interdits.
- **Code Names** : Jeu d'association de mots en équipes.
- **Loup-Garou** : Le grand classique des jeux de rôles cachés.
- **Ranking** : Qui est le plus... ? Classez vos amis !
- **BMC** : Un jeu de cartes drôle pour compléter des phrases.


### Lancer l'Application (Frontend)
Allez dans le dossier frontend :

   cd frontend
   npx expo start --go
   

---

### 4. Connecter votre Téléphone
1. Un **QR Code** va s'afficher dans votre terminal frontend.
2. Ouvrez l'application **Expo Go** sur votre téléphone.
3. Scannez le QR Code.
4. L'application va se charger (cela peut prendre quelques secondes la première fois).
